# Sorteggiatore-per-Consegne
Uno strumento leggero in Python progettato per semplificare la gestione delle consegne della Croce Rossa. Il software organizza gli indirizzi, gestisce le liste di distribuzione e calcola i percorsi ottimali partendo da un punto di origine.

📋 Funzionalità
Gestione Anagrafica: Salvataggio di nomi, cognomi, indirizzi e note specifiche in formato JSON.

Filtro Consegne: Identificazione rapida degli utenti inclusi nella lista di consegna attuale.

Ottimizzazione Percorsi: Suddivisione automatica degli indirizzi in base al numero di giri di consegna necessari.

Ordinamento Intelligente: Calcolo dell'ordine ottimale delle tappe per minimizzare i tempi di percorrenza dal nodo di origine.

🚀 Installazione e Uso
Assicurati di avere Python 3.x installato.

Clona il repository o scarica i file.

Configura il file JSON con i dati necessari.

Avvia lo script principale per generare i giri di consegna.
